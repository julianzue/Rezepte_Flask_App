# Rezepte_Flask_App

## Beschreibung
Diese Flask App, geschrieben in Python, kann Rezepte speichern und aufbewahren.

## Installation (Docker)
- sudo docker build -t rezepte .
- sudo docker run --restart unless-stopped -p 5000:5000 -d rezepte
